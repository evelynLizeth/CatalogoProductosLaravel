# CatalogoProductosLaravel

## REQUISITOS TÉCNICOS PREVIOS
- Instalar PHP (versión 8.1 o superior recomendada).
- Instalar Composer (el gestor de dependencias de PHP).
- Instalar Laravel.
- Servidor local: Laragon.
- Base de datos: MySQL.
- Editor de código: Visual Studio Code.
- Git: para control de versiones.

## PASOS PARA INSTALAR DEPENDENCIAS
1. **Instalar dependencias de Laravel**  
   En la carpeta del proyecto: `C:\laragon\www\CatalogoProductosLaravel` abrir la terminal y ejecutar:  php composer.phar install
2.	**Generar la clave de la aplicación:** (php artisan key:generate)

3.	**Levanta el servidor:** (php artisan serve)

4.	**Abrir el navegador:** http://127.0.0.1:8000
